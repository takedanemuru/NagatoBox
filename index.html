Requirement

All softwares of NagatoBox are made by Gambas3, and require Gambas3 3.5.2 or later.
Ubuntu official packages are too old to run NagatoBox. use PPA instead. 

$ sudo add-apt-repository ppa:nemh/gambas3
See Gambas3 official page to learn more.
